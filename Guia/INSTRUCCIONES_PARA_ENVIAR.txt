🔧 SI TU AMIGO TIENE PROBLEMAS
═══════════════════════════════════════════════════════════════════

Los problemas más comunes (todos explicados en la guía):

1. Puerto 3000 ocupado → Cambiar puerto en backend/index.js
2. MySQL no conecta → Verificar que esté corriendo en XAMPP
3. npm install falla → Verificar versión de Node.js (v20+)
4. Frontend en blanco → Verificar que backend esté corriendo
5. Módulos no encontrados → Ejecutar npm install en backend Y frontend
