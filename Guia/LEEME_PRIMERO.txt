📋 ¿QUÉ ES ESTO?
═══════════════════════════════════════════════════════════════════

Esta es una aplicación web COMPLETA de recetas de postres.
Incluye:
✅ Frontend en React 19
✅ Backend en Node.js + Express
✅ Base de datos MySQL
✅ Sistema de usuarios (admin, editor, usuario)
✅ Crear, editar, eliminar recetas
✅ Sistema de favoritos y colecciones
✅ Comentarios y valoraciones (estrellas)
✅ Notificaciones en tiempo real
✅ Panel de administración con gráficas animadas
✅ Diseño responsive y moderno

🎯 RESUMEN RÁPIDO (si ya tienes experiencia)
═══════════════════════════════════════════════════════════════════

Si ya tienes Node.js v20+ y XAMPP instalados:

1. Importar database/schema_completo.sql en phpMyAdmin
2. cd backend && npm install && npm start
3. cd frontend && npm install && npm run dev
4. Abrir http://localhost:5173
