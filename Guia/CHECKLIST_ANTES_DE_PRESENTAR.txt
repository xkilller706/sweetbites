🚨 PROBLEMAS COMUNES Y SOLUCIONES
─────────────────────────────────────────────────────

Si algo falla en la presentación:

❌ "No carga la página"
   → Verificar que backend esté corriendo
   → Verificar que MySQL esté verde

❌ "Error de conexión"
   → Reiniciar backend (Ctrl+C, npm run dev)
   → Verificar que BD existe

❌ "No se ven las imágenes"
   → Las de Unsplash requieren internet
   → Usar fotos locales si no hay internet

❌ "No puedo registrarme"
   → Verificar que la BD esté importada
   → Verificar consola del navegador (F12)
