PRÓXIMOS PASOS
─────────────────────────────────────────────────────

1. ✅ Instalar Node.js
2. ✅ Instalar XAMPP
3. ✅ Crear base de datos en phpMyAdmin
4. ✅ Importar schema.sql
5. ✅ Ejecutar: npm install (en backend/)
6. ✅ Ejecutar: npm run dev (en backend/)
7. ✅ Abrir index.html en navegador
8. ✅ ¡Disfrutar de SweetBites!
